MLB Shortstop Analysis

This dashboard supports our project goal of identifying the most suitable shortstop replacement when a player is unavailable. Using 2025 player performance, risk indicators, clustering-based archetypes, and a pairwise recommendation table, the page helps compare candidates by age, contract level, injury history, cluster profile, and on-field value to find practical replacement options.
